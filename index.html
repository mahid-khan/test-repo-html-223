In other words: summing up some numbers. $$\sum_{i=1}^{100}i$$ Here, `i=1` says to start at `1` and end at the number above the Sigma, `100`. These are the lower and upper bounds, respectively. The *i* to the right of the "E" tells us what we are summing. In code: ```js var sum = 0 for (var i = 1; i <= 100; i++) { sum += i } ``` The result of `sum` is `5050`. --- # Python If you don't know any Python, you will need to learn some.
